# Shell - Basics

### Description
This repository contains files used for learning Shell basics via the ALX SE Programme. 

### Requirements
* Interpreter: Shell
* OS: Ubuntu 20.04 LTS
* Editors: ```vi```, ```vim```, or ```emacs```.

### Concepts
* What is the Shell.
* Navigation through Linux systems
* Manipulating files
* Working with commands
* Reading Man pages
* Keyboard shortcuts for Bash
* LTS
* Shebang

### SCRIPTS' FUNCTION

1. **0-current_working_directory** prints the absolute path name of the current working directory.<br>
2. **1-listit** displays the contents of the working directory.<br>
3. **2-bring_me_home** writes a script that changes the working directory to the user's home directory.<br>
4. **3-listfiles** displays current directory contents in a long format.<br>
5. **4-listmorefiles** displays current directory contents, including hidden files.<br>
6. **5-listfilesdigitonly** displays current directory contents with user and group IDs displayed numerically.<br>
7. **6-firstdirectory** creates a directory in another directory.<br>
8. **7-movethatfile** moves files between directories.<br>
9. **8-firstdelete** deletes a file called betty from a specific directory.<br> 
10. **9-firstdirdeletion** deletes a certain directory located in another directory.<br>
11. **10-back** changes the working directory to the previous one.<br>
12. **11-lists** list all files/folders (even the hidden ones) from multiples directories.<br>
13. **12-file_type** prints the file type of a certain file.<br>
14. **13-symboli_links** creates a symbolic link in the current working directory.<br>
15. **14-cpy_html** copy html files from one directory to the other without overwriting files that already exist.<br>
16. **100-lets_move** move all files beginnin with upper case letters from working directory to a specific directory.<br>
17. **101-clean_emacs** delete all files in the working directory that end with a ~.<br>
18. **102-tree** creates multiple directories in one line.<br>
19. **103-commas** lists all the files and directories of the current directory, separated by commas.<br>
20. **school.mgc** elaborates on the use of *type* command.<br>
