## SCRIPT'S DESCRIPTION

1. **0-alias** creates an alias named "ls".<br>
2. **1-hello_you** prints "hello user" where user is the current Linux user.<br>
3. **2-path** adds /action to the PATH. /action is the last directory the shell looks into when looking for a program.<br>
4. **3-paths** counts the number of directories in the PATH.<br>
5. **4-global_variables** lists the environment variables.<br>
6. **5-local_variables** lists all local and environment variables, and functions.<br>
7. **6-create_local_variable** creates a new local variable.<br>
8. **7-create_global_variable** creates an new global variable.<br>
9. **8-true_knowledge** prints the results of the addition of 128 with the value stored in the environment variable TRUEKNOWLEDGE.<br>
10. **9-divide_and_rule** prints the results of POWER divided by DIVIDE, followed by a newline.<br>
11. **10-love_exponent_breath** display the result of BREATH to the power LOVE.<br>
12. **11-binary_to_decimal** converts an number from base 2 to base 10.<br>
13. **12-combinations** prints all possible combinations of 2 letters, except "oo".<br>
14. **13-print_float** prints a number with 2 decimal places followed by a newline.<br>
15. **100-decimal_to_hexadecimal** converts a number from base 10 to base 16.<br>
16. **101-rot13** encodes and decodes text using rot13 encryption.<br>
17. **102-odd** prints every other line from the input, starting with the first line.<br>
18. **103-water_and_stir** adds the 2 numbers stored in the environment variables WATER and STIR and prints the result.<br>
